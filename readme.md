# Passerelle

Passerelle micro:bit - Raspberry - API avec MQTT

## Dependencies
  - Python 3.8+
  - [Requirements file]()

```sh
apt install git python3 python3-pip python3-venv
```
## Setup
```sh
git clone https://github.com/CPE-2023-Projet-SDMIS/Passerelle-simulateur.git
cd Passerelle-simulateur
./setup.sh
```

## on raspberry pi :
```sh
./install-service.sh
```

## Start
```sh
./start.sh
```
