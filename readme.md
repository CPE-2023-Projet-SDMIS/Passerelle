# Passerelle

Passerelle micro:bit - Raspberry - API avec MQTT

## Dependencies
  - Python 3.8+
  - [Requirements file]()

## Setup
```sh
git clone https://github.com/CPE-2023-Projet-SDMIS/Passerelle-simulateur.git
cd passerelle
./setup.sh
```

## on raspberry pi :
```sh
./install-service.sh
```

## Start
```sh
./start.sh
```
